# 📚 Intro to Computer Science ~ CS1

> My first ever CS course, taken during my freshman year of college. This repository contains all the Java programs I wrote while learning the fundamentals of programming from scratch.

---

## 🧑‍💻 About This Course

This was my introduction to computer science and object-oriented programming using **Java**. The course covered core programming concepts through hands-on projects ranging from drawing shapes to managing collections of data.

---

## 🗂️ Topics Covered

| Topic | Description |
|-------|-------------|
| **Shapes** | Drawing and calculating geometry using classes and objects |
| **Songs** | Looping and string manipulation through song-lyric programs |
| **Arrays** | Storing and iterating over fixed-size collections of data |
| **ArrayLists** | Dynamic lists with add, remove, and search operations |
| **Classes & Objects** | Building custom types with fields and methods |
| **Scanners** | Reading user input from the console |
| **Math & Expressions** | Arithmetic, conditionals, and logical expressions |
| **Greetings & People** | Intro to OOP with simple real-world models |
